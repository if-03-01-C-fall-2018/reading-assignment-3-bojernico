HTML for Lists

Start tags for:

  • an ordered list: ol

  • an unordered list: ul

  • a description list: dl

  • a list item: li

  • a description term: dt

  • a description data: dd

CSS for Lists

  • Make bullet a square: list-style-type: square
  • Make bullet an image named "my-bullet.png": list-style-image: url('my-bullet.png');
  • Make bullet an uppercase roman number: list-style-type: upper-roman
  • Set bullet position to be inside the list item's text flow: list-style-position: inside
  • Set bullet position to be outside the list item's text flow: list-style-position: outside

HTML for Tables

Start tags for

  • a table: table
  • a table row: tr
  • a table header: thead
  • a table header cell: th
  • a table body: tbody
  • a table cell: td
  • a table footer: tfoot
  • a table footer cell: tf

CSS for Tables

Black dotted border of 1px width

table,th,td { border: 1px solid black; }

Solid border of 1px width having a red color and rounded corners

border: 1px red; border-radius: 5px;

Merge the borders of table cells so that they appear as one line: border-collapse: collapse;
